# Анализ данных интернет-магазина

**Цель проекта:**  
Провести комплексный анализ данных интернет-магазина: оценить качество данных, выявить паттерны покупательской активности, провести RFM-сегментацию клиентов и проверить статистические гипотезы.

**Инструменты:**  
Python (pandas, numpy, matplotlib, seaborn, scipy, statsmodels), Jupyter Notebook

## Ключевые результаты

### 1. Качество данных
- Данные за 2018 год признаны нерепрезентативными (334 дня без продаж)
- Для анализа использован период декабрь 2018 – декабрь 2019

### 2. Покупательская активность
- Пик заказов: **12–13 часов дня**
- Самый активный день: **воскресенье**
- Ночью (21:00–6:00) заказов нет

### 3. RFM-сегментация
- VIP-клиенты (21.6% базы) приносят **70.9% выручки**
- Лояльные клиенты (6%) — ещё **8% выручки**
- Спящие и разовые клиенты (44.2%) дают лишь **13.1% выручки**

### 4. Сравнение возвратных и разовых клиентов
- Возвратные клиенты (64.2% базы) тратят в **9 раз больше** разовых (225 тыс. vs 24 тыс.)
- Средний чек у обеих групп одинаков (~3000 руб.)

### 5. Статистические гипотезы
- **Гипотеза 1:** доля возвратных клиентов стабильна между кварталами (p-value = 0.44)
- **Гипотеза 2:** средний чек в стране 24 значимо выше, чем в стране 3 (p-value = 0.01)
- **Гипотеза 3:** количество товарных позиций в стране 5 выросло в Q4 (p-value < 0.05)

## Рекомендации для бизнеса

- **VIP-клиенты:** эксклюзивные предложения, индивидуальные скидки
- **Лояльные клиенты:** программа лояльности, бонусы за рекомендации
- **Постоянные и перспективные:** акции для повышения частоты покупок
- **Спящие и разовые:** reactivation-кампании, спецпредложения
